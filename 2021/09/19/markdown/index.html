markdown

首行缩进

   markdown不支持自动首行缩进，可通过 &ensp;实现。每次输入一个 &ensp;，都代表缩进一字符。

换行

   段落换行，须在端末后留下两个或以上空格，如：

无空格

会当临绝顶，

一览众山小。

有空格

会当临绝顶，

一览众山小。

插入网址

   使点击的词自动跳转到某一网站，如百度。语法为

[文字](网址)
